InnoDB):- скорость IO важнее сохранности данных;- нужна компрессия таблиц для экономии места на диске;- размер буффера с незакомиченными транзакциями 1 Мб;- буффер кеширования 30% от ОЗУ;- размер файла логов операций 100 Мб.Приведите в ответе изменённый файл `my.cnf`.```# For advice on how to change settings please see# http://dev.mysql.com/doc/refman/8.2/en/server-configuration-defaults.html[mysqld]## Remove leading # and set to the amount of RAM for the most important data# cache in MySQL.
